워드 양식 일괄 생성 실습자료

구성 파일
1. 참석자명단.xlsx
   - 참석자별 이름, 부서, 직급, 교육일자, 과정명, 교육장소가 들어 있습니다.

2. 공문양식_템플릿.docx
   - {{이름}}, {{부서}}, {{직급}}, {{교육일자}}, {{과정명}}, {{교육장소}} 자리표시자가 들어간 워드 템플릿입니다.

3. 워드_일괄생성_코드.py
   - 엑셀 명단을 읽어 참석자별 워드 문서를 자동 생성하는 예시 코드입니다.

4. 생성결과_예시 폴더
   - 코드 실행 결과 예시 워드 파일입니다.

필요 라이브러리
pip install pandas openpyxl python-docx

실행 방법
1. 이 폴더에서 터미널 또는 명령 프롬프트를 엽니다.
2. python 워드_일괄생성_코드.py 를 실행합니다.
3. 생성결과 폴더에 참석자별 공문 파일이 생성됩니다.
